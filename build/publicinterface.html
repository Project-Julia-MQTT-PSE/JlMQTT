JlMQTT.MqttConnect — Method.
JlMQTT.MqttConnect(client::JlMQTT.MqttClient, clientId::String)

Sends CONNECT message to broker and receives CONNACK message from broker.

Parameters:

client - [in] JlMQTT.MqttClient

clientId - [in] client identifier of type String

username - [optional] type String

password - [optional] type String

will - [optional] JlMQTT.WillOptions

willFlag - [optional] will flag of type Boolean

cleanSession - [optional] clean session flag of type Boolean

Returns:

[out] return code received from broker CONNACK message
